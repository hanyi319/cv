点滴记账简洁版

项目预览 项目源码
  • 一款基于 Vue 2 并使用 Vue CLI 构建的移动端本地记账应用。
  • 本项目构建形式为 SFC 单页面应用，并使用 Vue Router 管理路由。主要功能为记账（选择收支类别、标签、日期、输入备注和金额）、管理标签（添加、改名、删除）、查看明细、统计展示（使用 ECharts 实现）。
  • 本项目使用了 Element UI 的日期选择器，以及 Day.js，并使用了 Lodash 做基数排序从而构造数据提供给 ECharts 进行显示。
  • 本项目关于全局状态管理进行了数次重构，从最开始全部挂载到 Window 上，再到 自己封装 store，最后使用 Vuex，并且记账数据保存至 localStorage。
  • 通过该项目让我初次体会到了使用框架搭建开发的过程，并首次使用 TypeScript 编写代码，但由于 Vue 2 对 TS 的支持不够完美，需要大量使用装饰器语法来配合构建 Vue 组件。
